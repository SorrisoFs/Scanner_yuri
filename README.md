1 - O problema do Threshold Comum: Se você usar o threshold comum de valor fixo em uma foto com sombras, 
a parte da folha que está na sombra vai ficar completamente preta, apagando o texto que estava ali.

2 - A Solução com Threshold Adaptativo: Usei um algoritmo inteligente que calcula um limite de corte diferente para cada região da folha. Onde está iluminado, o corte é alto; onde tem sombra, o corte cai automaticamente.
Isso permite isolar o texto perfeitamente, não importa quão ruim esteja a iluminação da foto.
